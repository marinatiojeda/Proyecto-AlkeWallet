# Descripción

Este proyecto consiste en el diseño e implementación de una base de datos relacional para el sistema **Alke Wallet**, desarrollada en **MySQL Workbench**.

Durante el desarrollo se aplicaron conceptos fundamentales de bases de datos, tales como:

- Creación de una base de datos relacional.
- Definición de tablas mediante sentencias DDL.
- Inserción de datos utilizando sentencias DML.
- Consultas SQL con SELECT.
- Actualización y eliminación de registros mediante UPDATE y DELETE.
- Implementación de transacciones con START TRANSACTION, COMMIT y ROLLBACK.
- Verificación de la integridad referencial mediante claves foráneas.
- Diseño del modelo Entidad-Relación (ER).
- Aplicación de los principios ACID.
